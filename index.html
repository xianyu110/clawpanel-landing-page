快速开始

只需几行命令，即可在本地启动 maynoraiclawpanel

$ git clone https://github.com/xianyu110/maynorai-clawpanel.git
$ cd maynorai-clawpanel && npm install
$ ./scripts/dev.sh
$ npm run tauri dev
