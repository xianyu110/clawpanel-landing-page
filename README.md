# ClawPanel Landing Page

OpenClaw AI Agent 可视化管理面板 - 产品介绍页面

## 预览

直接在浏览器中打开 `index.html` 即可查看

## 功能特性

- 🎨 现代深色主题设计
- 🌟 玻璃拟态 UI 风格
- 📱 响应式布局
- ⚡ 纯静态 HTML/CSS，无需构建

## 技术栈

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- 原生 JavaScript (无框架依赖)

## 本地预览

```bash
# 克隆仓库
git clone https://github.com/your-username/clawpanel-landing.git

# 进入目录
cd clawpanel-landing

# 用浏览器打开
open index.html
```
